*
*
*	FOLLOW STEPS TO INSTALL
*
*

1. Go to Kernel FOLDER in taulop_toolbox/taulop_lib
2. ./configure
3. make
4. sudo make install
5. Add your default library folder (usually /usr/local/lib) to LD_LIBRARY_PATH
	5.1 LD_LIBRARY_PATH=/usr/local/lib (your default library path)
	5.2 export LD_LIBRARY_PATH


Now you can use library and run tests with -ltaulop
